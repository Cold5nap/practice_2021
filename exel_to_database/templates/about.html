Возможности сайта:

  • Преобразует загруженные Excel файлы в sqlite, sqlite.txt, mysql.txt, где
    sqlite - база данных, имеет формат ".db",
    sqlite.txt - текстовый файл, в котором предоставлены команды для построения базы данных sqlite,
    mysql.txt - текстовый файл, в котором предоставлены команды для построения базы данных sqlite
  • Редактор дает возможность установить название аттрибута, его типа, ограничений,
    установить является ли атрибут внешним ключом для определенной таблицы и атрибута
  • В сайте предусмотрена возможность хранения файлов excel для их дальнейшего преобразования

Описание ролей:

  • Участник - пользователь, имеющий возможность преобразования файлов excel в базу данных и хранить файлы excel
  • Администратор - пользователь, имеющий возможность управлять всеми зарегистрированными пользователями,
    а также хранящимися файлами excel
